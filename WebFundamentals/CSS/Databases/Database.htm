What are Databases?

A structured set of data

Databases are a collection of information, they are used for creating,manipulation, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or a non-relational database. Relationships means a set of our data is relationed to another section of our data. Meanwhile in a non-relational database there is not a common related section of our data. Our databases are put into sctions of data called schemas, inside each schema each data set is organized into tables, those tables contain columns and rows to house that piticular set of data.

Storage

The amaxing part about having a Database is that it is a storage bin that holds onto all saved data.
