What is Git?

Git is a version control system; you can think of it as a seies of snapshots(commits) of your code base. With the creation of each snapshot you build out a whole application,seeing the growth of a full app. During this time you can make branches to experiment and come back to a previous snapshots you took or merge those experiments into yout app.

Git add

Git add is how we stage the intergration of our current code base into our current list of snapshots.

Git status

Git status allows us to see the changes we are going to make.

Git commit

Git commit; finally we can commit those changes we added to our last snapshot giving us a new snapshot of our code with whatever new changes were made.

Git remote add

Git remote add is used to set up an online repoisitory where we can place all of our current code. This is where the outside world can see our code base.
